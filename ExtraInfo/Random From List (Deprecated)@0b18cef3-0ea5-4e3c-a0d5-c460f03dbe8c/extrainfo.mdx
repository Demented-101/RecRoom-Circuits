### Issues

_Oh boy this one is a doozy!_

This chip would work by changing its output on each call, so to receive an element did not require an exc, however two sequenced chips would not receive the same output if the chip was not origionally executed.<br>
The Exc pin on this chip does practically nothing other than lock the output for a single tick. This meant that the output was not saved between ticks unlike any other chip.
